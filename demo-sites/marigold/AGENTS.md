# Marigold website editing rules

- This directory is the entire website workspace. Do not read or modify files outside it.
- Content shown in the Snoball preview lives in `content.json`.
- Preserve the existing content keys unless the user explicitly requests a structural change.
- Make the smallest complete change that satisfies the request.
- Never deploy, publish, access credentials, or make network requests.
- Summarize what changed and name the files you edited.
